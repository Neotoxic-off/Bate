## Usage
```BASH
Usage: bate [OPTIONS] --file <FILE> --keys <KEYS>

Options:
  -f, --file <FILE>
  -k, --keys <KEYS>
  -s, --score <SCORE>      [default: 0.7]
  -h, --help               Print help
```
